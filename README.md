# SoundPod-LRC

This repository is the central database for synchronized lyrics used by the [SoundPod](https://github.com/arunnechully/SoundPod) music player.

### How it works:
* **Open Source**: All lyrics are stored in the standard `.lrc` format.
* **Transparent**: No user data is tracked; we only sync song IDs and lyrics.

### Structure:
All synced files are located in the `/lyrics` folder, named by their unique Media ID.
